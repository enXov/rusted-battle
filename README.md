# Rusted Battle

A 2D multiplayer arena brawler built from scratch in Rust, inspired by Bopl Battle.

## Description

Rusted Battle is an open-source arena brawler where players battle using unique characters with diverse abilities. Built entirely from scratch using Rust and low-level libraries, the game prioritizes performance, memory safety, and cross-platform compatibility.

## Features

- **Pure Rust**: Built from the ground up using Rust
- **Zero Memory Leaks**: Leveraging Rust's safety guarantees
- **Cross-Platform**: Native builds for Windows and Linux
- **Open Source**: Free forever, community-driven development

## Getting Started

### Prerequisites

- Rust (latest stable) - [Install Rust](https://rustup.rs/)
- GPU with Vulkan, DirectX 12, or Metal support

### Building

```bash
# Clone the repository
git clone https://github.com/enxov/rusted-battle.git
cd rusted-battle

# Build and run
cargo run --release
```

### Development

```bash
# Run in development mode
cargo run

# Run tests
cargo test

# Check code
cargo clippy
```
